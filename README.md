## 🚀 Quick Overview

### 1️⃣ NLP Text Summarizer
> Extractive + abstractive summarisation pipeline using Facebook BART and DistilBERT sentiment analysis.
```bash
cd 1-nlp-text-summarizer && pip install -r requirements.txt && python summarizer.py
```

### 2️⃣ Computer Vision Classifier
> Transfer learning (ResNet-50) and custom CNN on CIFAR-10 with Grad-CAM explainability.
```bash
cd 2-computer-vision-classifier && pip install -r requirements.txt && python classifier.py
```

### 3️⃣ House Price Predictor
> Full scikit-learn pipeline comparing 7 regression models with EDA and feature engineering.
```bash
cd 3-ml-house-price-predictor && pip install -r requirements.txt && python predictor.py
```

### 4️⃣ Deep Learning CNN
> Residual CNN with early stopping, label smoothing, confusion matrix, and t-SNE embeddings.
```bash
cd 4-deep-learning-cnn-mnist && pip install -r requirements.txt && python cnn_trainer.py
```
